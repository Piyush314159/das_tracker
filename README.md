# DSA Tracker 🔥

A self-updating DSA problem tracker — powered by GitHub Pages.

**Live site → https://Piyush314159.github.io/DSA_Tracker**

---

## How it works

1. Add a `.py` file to the right folder
2. Push to GitHub
3. The website **automatically shows your new problem** — no config needed

---

## File naming convention

```
<problem_name>_<difficulty>.py
```

| Example filename             | Shows as          | Difficulty |
|------------------------------|-------------------|------------|
| `two_sum_easy.py`            | Two Sum           | 🟢 Easy    |
| `longest_substring_medium.py`| Longest Substring | 🟡 Medium  |
| `median_two_arrays_hard.py`  | Median Two Arrays | 🔴 Hard    |

> If you forget the difficulty suffix it defaults to **medium**.

## Setup (one-time)

1. Create a new GitHub repo named `DSA_Tracker`
2. Upload all these files
3. Go to **Settings → Pages → Source → main branch → / (root)**
4. Your site will be live at `https://Piyush314159.github.io/DSA_Tracker`

> **Make sure the repo is PUBLIC** so the GitHub API can read your files without authentication.

---

## Solved state

Your solved/unsolved status is saved in your **browser's localStorage** — it stays even after refresh, but is local to your browser.
